# 파수의 등대 — Phase 2: saveticker 스타일 레이아웃 (디자인 ONLY)

## 목표
현재 단촐한 탭바+카드 구조를 saveticker 수준의 밀도감 있는 뉴스 플랫폼으로 업그레이드.
**이번 Phase는 디자인(HTML/CSS)만. 기능 구현은 NO.**
모든 버튼/드롭다운/필터는 시각적 배치만 하고 클릭 동작은 하지 않아도 됨.

---

## 새 레이아웃 구조 (위→아래)

### 1. 검색바 (신규)
```
┌─────────────────────────────────────────┐
│  🔍 검색어 입력...                       │
└─────────────────────────────────────────┘
```
- `index.html` 상단, 탭바 바로 아래
- 둥근 모서리(24px), 밝은 회색 배경(#f1f3f5), 🔍 아이콘 왼쪽
- placeholder: "검색어 입력..."
- 너비: max-width 720px, 가운데 정렬

### 2. 통합 필터바 (기존 필터 개선)
```
[★전체] [★★★] [★★] [★]   |   [종합] [속보] [분석] [반도체]
[🌐전체] [🇰🇷국내] [🌍글로벌] [💼내포트폴리오]
```
- 두 줄로 배치 (1행: 중요도 + 카테고리, 2행: 지역 + 포트폴리오)
- 활성화된 필터: 진한 배경(dark gray) + 흰 글씨
- 비활성: 밝은 회색 배경 + 짙은 회색 글씨
- 둥근 pill 모양 (border-radius: 20px), padding: 6px 14px
- 현재 ★★★/★★/★ 점수별, 전체/국내/글로벌 필터 기능은 그대로 유지
- 신규: [종합] [속보] [분석] [반도체] [스페이스X] [이란] 등 태그 추가 (클릭 동작 X, 디자인만)

### 3. 🔥 오늘 주요뉴스 섹션 (신규)
```
🔥 오늘 주요뉴스  ▸                              [접기▲]
┌──────────────────┐ ┌──────────────────┐
│ [SAVE·정보] 2시간전│ │ [SAVE·정보] 5시간전│
│ 제목...           │ │ 제목...           │
│              👁 27.7K│ │              👁 42.2K│
└──────────────────┘ └──────────────────┘
```
- 헤더: 왼쪽에 "🔥 오늘 주요뉴스 ▸", 오른쪽에 접기/펼치기 토글
- 2열 CSS Grid (grid-template-columns: 1fr 1fr, gap 12px)
- 각 카드: 제목(2줄 제한), 출처 태그(좌상), 상대시간(우상), 조회수(우하)
- 데이터: ★★★ 뉴스 중 상위 4개 사용 (기존 로직)
- 모바일에서는 1열로 전환 (@media)

### 4. 뉴스피드 헤더 (기존 개선)
```
📰 뉴스                          [최신순 ▾] [🔄]
```
- 왼쪽: "📰 뉴스"
- 오른쪽: 정렬 드롭다운 "최신순 ▾" + 새로고침 버튼 🔄
- 드롭다운: 둥근 테두리, 클릭 시 "최신순 / 조회순 / 평점순" 보이게 (디자인만, JS 동작 X)

### 5. 뉴스피드 카드 (기존 개선)
```
┌─────────────────────────────────────────────┐
│ [로이터·시황]                    8분 전      │
│ FOREX-Yen teeters on cusp of 40-year low...  │
│                                    👁 59     │
└─────────────────────────────────────────────┘
```
- 각 카드: 출처·카테고리 태그(좌상), 상대시간(우상), 제목(좌), 조회수(우하)
- 조회수: `view_count` 필드 표시 (없으면 0으로)
- 기존 별점(★) 표시 유지

## CSS 가이드

### Design Tokens (tokens.css에 추가)
```css
--search-bg: #f1f3f5;
--filter-active-bg: #333;
--filter-active-text: #fff;
--filter-inactive-bg: #f1f3f5;
--filter-inactive-text: #555;
--card-shadow: 0 1px 3px rgba(0,0,0,0.06);
--card-hover-shadow: 0 4px 16px rgba(248, 187, 208, 0.2);
```

### 기존 디자인 유지
- body: 파스텔 핑크 물결 배경 (건드리지 말 것)
- 히어로 섹션: transparent 배경 (건드리지 말 것)
- 전체 폰트, 컬러, 파스텔 핑크 테마 유지
- 네비게이션 탭바 (뉴스|포트폴리오|캘린더|지표): 그대로

### 반응형
- 모바일(@media max-width 768px): 주요뉴스 1열, 검색바 full-width
- 데스크탑: max-width 960px 컨텐츠 영역, 가운데 정렬

## 수정 파일

1. **`templates/index.html`** — 검색바, 필터바, 주요뉴스 2열, 뉴스피드 헤더/카드 구조
2. **`static/style.css`** — 해당 섹션 스타일 추가 (기존 body/hero/theme 유지)
3. **`static/tokens.css`** — 신규 토큰 추가

## 중요 규칙

1. **디자인 ONLY** — JavaScript/AJAX/폼 submit 등 기능 구현 금지
